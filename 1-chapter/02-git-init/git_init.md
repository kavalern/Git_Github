### Створення репозиторію. 

Команда `git init` створює новий локальний репозиторій Git у поточній директорії, ініціалізуючи приховану папку .git для зберігання всієї службової інформації

```bash
# створити каталог для проєкту
mkdir folder_name

# вивести вміст поточного каталогу
ls

# вивести шлях до поточної директорії
pwd

# вивести вміст каталогу в детальному (табличному) вигляді, включно з прихованими файлами та папками
ls -la

# створити Git-репозиторій у поточній папці
git init

# вивести вміст файлу config
cat config

# вивести список глобальних налаштувань Git
git config --global --list

# вивести список локальних налаштувань Git
git config --local --list

# повернутися до попередньої директорії
cd -

# перейти до домашнього каталогу користувача
cd ..
```
