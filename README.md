# Вивчення Git та використання Github/Gitlab

### **Git – Система контролю версій.**

Існує не одна система контролю версій, VCS (Version Control System) вони відрізняються за типами – які в свою чергу корелюються взалежності від площі використання системи контролю задач. 

Глибше занурюватись тут не буду, оскільки наразі мене цікавлять розподілені системи котролю версій – яким і є Git в свою чергу. 

***Тезисно, спробую описати для чого необхідна система контролю версій:***

* Мати одночасно декілька версії коду,відслідковувати комміти (зміни) протягом певного тайм-лайну

* Структуризація. Кожен комміт/зміна закріплена за автором, часом.

Також можна порівнювати кожен комміт щоб бачити зміни у коді 

* Можливість відкотитись до будь-якої версії проекту

**Система контролю версії найбільше розповсюджена для командної роботи**
